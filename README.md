# PMDM-Mapas

Este proyecto es una aplicación móvil desarrollada en **Kotlin** como parte del aprendizaje en la asignatura de Programación de Dispositivos Móviles (PMDM). La aplicación está centrada en el uso de mapas, permitiendo a los usuarios interactuar y visualizar ubicaciones geográficas de manera dinámica.

## Características principales

- Integración con mapas interactivos (por ejemplo, Google Maps o Mapbox).
- Visualización de ubicaciones y marcadores en el mapa.
- Posibilidad de centrar el mapa según la ubicación actual del usuario.
- Interfaz sencilla, ligera y adaptada a dispositivos móviles Android.

## Tecnologías utilizadas

- **Kotlin:** Lenguaje principal de desarrollo de la aplicación.
- **Android SDK:** Para la creación de la app y manejo de componentes nativos de Android.
- **Biblioteca de mapas:** (especifica aquí, por ejemplo: Google Maps API, Mapbox, etc.).

## Instalación y uso

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/inndiaass/PMDM-Mapas.git
   ```

2. **Abre el proyecto en Android Studio.**

3. **Configura las claves necesarias** (por ejemplo, API Key para Google Maps) en el archivo adecuado (por ejemplo, `local.properties` o el archivo correspondiente en `res/values`).

4. **Compila y ejecuta la aplicación** en un emulador o dispositivo Android.
